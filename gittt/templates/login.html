What is a Personal Access Token?

A Personal Access Token (PAT) is an alternative to using passwords for authentication to GitHub.

  1. Go to your GitHub account settings
  2. Select "Developer settings" from the sidebar
  3. Click on "Personal access tokens" → "Tokens (classic)"
  4. Click "Generate new token" → "Generate new token (classic)"
  5. Give your token a name (e.g., "GitHub GUI Manager")
  6. Select the following scopes:
    • repo (all)
    • workflow
    • read:user
  7. Click "Generate token"
  8. Copy your token and paste it in the next screen

⚠️ Keep your token secure! Treat it like a password.
